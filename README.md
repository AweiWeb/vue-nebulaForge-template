# vue-nebulaForge-template
一个从0搭建的vue-template

💣注意写vue3的时候不能使用ts，这里还没没有配置vue3 + ts的解析的插件，所以无法解析
这里会报错💣

使用组合式API写法
例如：
<script>
import { ref } from 'vue';
export default {
    setup() {
        const ta = ref('aaaa')
        console.log(ta.value)
    }
}
</script>
